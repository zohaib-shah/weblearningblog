- **GitHub Pages**: https://zohaib-shah.github.io/weblearningblog/

## Structure

- `_posts/` - Blog posts organized by date (YYYY-MM-DD-title.md)
- `docs/` - Alternative organization by category folders
- Categories: Angular, AngularJS, AWS, CI/CD, Django, JavaScript, Laravel, Node.js, PHP, TypeScript

## Local Development

```bash
# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve

# Visit http://localhost:4000/weblearningblog/
```
